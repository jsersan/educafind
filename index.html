educafind

Encuentra centros educativos en Euskadi

educafind es un sitio web diseñado para buscar centros educativos en todo Euskadi. Este sitio web es un proyecto educativo como alternativa de código libre al buscador de centros de Ikasgunea. Es una página creada totalmente en código abierto, sin ánimo de lucro y puede modificarse por cualquier persona siempre y cuando respete la licencia Creative Commons.

Este sitio web está totalmente diseñado de cara a front-end, por lo que no necesita conectarse a ninguna base de datos. Solamente necesitas un ordenador, un navegador y ya puedes arrancar con educafind.

Olvídate de usar Google Maps para desarrollar y pásate a OpenLayers. Ofrece una gran cantidad de personalización gracias a la función de capas, además de no requerir una clave de pago para distribución de páginas web.
